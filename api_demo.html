API Overview

This section showcases the API endpoints available on this portfolio website. These APIs provide data about my skills, experience, and projects, and demonstrate my backend development capabilities. Feel free to interact with them using the buttons below.

All endpoints return JSON data and follow RESTful principles. These APIs are implemented using Flask and serve as practical examples of my integration and data engineering skills.

Skills API

This API returns a comprehensive list of my technical skills categorized by domain, along with proficiency levels.

GET /api/data/skills

Experience API

This API returns details about my professional experience, including positions, companies, and responsibilities.

GET /api/data/experience

Weather API Demo

This is a demonstration API that returns weather data for a specified city.

GET /api/demo/weather?city={city_name}

GitHub Repositories API

This API fetches and returns my GitHub repositories data, demonstrating integration with external APIs.

GET /api/github/repos

This endpoint fetches data from the GitHub API and returns it in a structured format. To see this in action, visit the GitHub page of this portfolio.
